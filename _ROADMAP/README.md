# _ROADMAP — crossroads swift

Forward-looking, committed scope. Distinct from `_RESEARCH/` (exploratory) and `_RETEX/` (retrospective).

## Contents

- `north-star.md` — 6-month horizon, the bet we're making
- `next-release.md` — what ships next (must / should / nice)
- `backlog.md` — everything not in next-release, prioritized

## Flow

```
_RESEARCH/ (explore) → backlog.md (prioritize) → next-release.md (schedule) → _CODE/ (build) → CHANGELOG.md (release) → _RETEX/ (lessons)
```

## Cadence

- **Weekly** — review `next-release.md`, cross off done, reshuffle the rest
- **Monthly** — re-prioritize `backlog.md`, drop stale items, archive shipped
- **Quarterly** — revisit `north-star.md`, update or rewrite

## Rule

A backlog item older than 120 days with no movement = cut it (or promote with justification). Stale backlogs are morale tax.
