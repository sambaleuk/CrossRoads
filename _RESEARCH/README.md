# _RESEARCH — crossroads swift

Exploratory discovery — things we might build, market intel, user interviews, competitive analysis. Distinct from `_ROADMAP/` (committed) and `_RETEX/` (retrospective).

## Contents (grow as needed)

- `user-interviews/` — transcripts + notes
- `competitive-landscape/` — who else is in this space
- `market-intel/` — industry reports, pricing data
- `feature-probes/` — write-ups of ideas not yet ready for `_ROADMAP/backlog.md`

## Rule

Research is a compression input. When an insight here materializes into a decision, it graduates to `_ROADMAP/backlog.md`.
